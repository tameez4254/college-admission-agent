# AI-Powered College Admission Agent Using RAG and IBM Granite

## 📌 Project Overview
The AI-Powered College Admission Agent is an intelligent chatbot developed using Retrieval-Augmented Generation (RAG), IBM Granite models, and FAISS vector database to provide accurate and real-time admission assistance to students. The system retrieves information related to courses, eligibility, fee structures, admission procedures, and FAQs from institutional documents and databases.

This project was developed as part of the IBM Edunet Virtual Internship.

---

## 🚀 Features
- Real-time admission assistance
- AI-powered chatbot interface
- RAG-based information retrieval
- IBM Granite model integration
- Semantic search using FAISS
- 24/7 student support
- Accurate and context-aware responses
